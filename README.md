# Physics-Prior Residual-Learning Surrogate for Concrete Compressive Strength

Code and data for the paper:

**"Reliability-based mix design for concrete compressive strength using a physics-prior residual-learning surrogate with calibrated uncertainty"**

## Dataset

The publicly available UCI Concrete Compressive Strength dataset (Yeh, 1998) containing 1,030 samples and 8 input variables is used. The dataset is located in `data/concrete_data.csv`.
